# 한국영화 영향력 지도

오마주, 감독의 직접 언급, 평론의 계보를 방향 그래프로 연결해 한국영화의 영향력을 탐색하는 정적 웹사이트입니다.

## 현재 범위

- KOFA 100선 가운데 허브 30편에서 출발한 파일럿 v1
- 연구 엣지 87개
- 대표 순위에 포함되는 검증 엣지 79개
- 공개 그래프 노드 87개
- 감독 직접 언급, 피인용 수, PageRank 순위
- 영화별 영향 관계와 원문 출처

이 결과는 KOFA의 공식 영향력 순위가 아닙니다. 수집 범위, 연식, 기록 가시성의 영향을 받습니다.

## 개발

Node.js 24 LTS가 필요합니다.

```text
npm ci
npm run validate:data
npm run test
npm run check
npm run build
npm run verify:dist
```

개발 서버는 프로젝트 규칙에 따라 백그라운드로 실행합니다.

```text
npm run dev
npm run dev:status
npm run dev:stop
```

## 데이터 흐름

```text
공개 스냅샷
  ↓ 스키마·참조 검증
정책 그래프 79개 엣지
  ↓ 지표 계산
87개 영화 상세 페이지와 순위·지도 생성
```

`랭킹_파일럿_v1.md`나 과거 HTML에서 숫자를 복사하지 않습니다. 모든 집계와 순위는 `data/public/dataset.v1.json`에서 빌드 시 다시 계산합니다.

## 공개 데이터

사이트의 `/data/`에서 JSON과 CSV를 받을 수 있습니다. 코드, 자체 편집 데이터, 제3자 발췌문의 권리는 서로 다릅니다. 데이터 재사용 라이선스는 아직 확정되지 않았습니다.

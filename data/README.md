# 공개 데이터 스냅샷

- `public/films.v2.json` - 영화 레지스트리. 안정 ID(film_0001), 제목·연도·감독, KOFA 100선 여부(상위 10편만 정확 순위), 공개 상태.
- `public/dataset.v2.json` - 영향 관계. 안정 ID(edge_0001), 방향(후대 → 선대), 관계 유형, 신호, 신뢰도, 공개 상태, 근거 발췌문(300자 이내), 출처 URL.

이 폴더는 `scripts/export-public-data.ts`(로컬 전용)로만 갱신한다. 손으로 고치지 않는다.
조사 원본, 검수 메모, 미검증 후보는 이 저장소에 포함되지 않는다.
